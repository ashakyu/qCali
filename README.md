# qCali
## 1.Project Description <br>
 -project 이름 : **qCali** <br>
 -project 소개 : 코로나 언택트 시대에 만남의 기회가 줄어들면서 소셜 네트워크가 '이야기'보다는 '사진'에 집중이 되는 것을 느낌. </br>
                본인만의 진솔한 이야기를 표현 할 수 있는 자유로운 커뮤니티 사이트의 필요성을 느끼고 제작
